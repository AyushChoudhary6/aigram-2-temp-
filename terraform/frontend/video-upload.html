API Endpoints

1. Get SAS Token

Generate a time-limited SAS token for secure upload to Azure Blob Storage.

POST /api/upload/sas-token
{ "fileName": "video.mp4", "userId": "user123" }

2. Upload Video to Blob Storage

Upload the video file directly to Azure Blob Storage using the SAS URL.

PUT {sasUrl} Headers: x-ms-blob-type: BlockBlob Content-Type: video/mp4

3. Store Metadata

Store video metadata in Cosmos DB for quick retrieval.

POST /api/upload/video { "blobName": "user123/...", "userId": "user123", "title": "My Video", "description": "...", "duration": 120, "tags": [] }

4. Get Metadata

GET /api/upload/metadata/:videoId?userId=user123

5. List User Videos

GET /api/upload/user/:userId/videos
